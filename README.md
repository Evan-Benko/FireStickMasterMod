# Wireless ADB FireStick Master Mod v2.0

## Intended Use — Start on Your Fire Stick

- Make sure your Fire Stick is in **Developer Mode** with **ADB Debugging ON**

### Enable Developer Mode
1. Go to:  
   **Settings → My Fire TV → About → Fire TV Stick**
2. Click **Fire TV Stick** **7 times** until it says:  
   *“You are already a developer”*

### Enable ADB Debugging
1. Go to:  
   **Settings → My Fire TV → Developer Options**
2. Set **ADB Debugging** to **ON**
